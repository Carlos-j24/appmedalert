🏥 MedAlert – Sistema de Alertas Médicas
🚀 Descripción

MedAlert es una aplicación web desarrollada para apoyar a los usuarios en la gestión de su salud, permitiendo el control de medicamentos, recordatorios médicos y citas programadas.

El sistema está diseñado para mejorar la adherencia a tratamientos médicos mediante alertas oportunas y una interfaz sencilla e intuitiva.

🎯 Objetivo del Proyecto

Desarrollar una aplicación web funcional que permita a los usuarios:

Gestionar sus medicamentos
Recibir alertas médicas
Registrar citas médicas
Tener control organizado de su tratamiento
